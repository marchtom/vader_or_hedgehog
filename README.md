Short script implementing AlexNet to distinguish pictures of Darth Vader from hedgehogs
(AlexNet: https://papers.nips.cc/paper/4824-imagenet-classification-with-deep-convolutional-neural-networks.pdf)

TFlearn, numpy, cv2
